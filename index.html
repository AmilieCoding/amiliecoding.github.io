These are my dotfiles, and they can be installed on all distros which support Hyprland! However, I do have to use the Nouveau drivers because NVIDIA drivers on Fedora are pretty awful so I am capped in what I can do games wise. So if you are going to use them, I heavily recommend using them on Arch or some other distro which has proper support for the NVIDIA drivers. I also recommend installing all the accompanied software like Mako, Waybar, Kitty and Wofi for the proper experience. If you would like to download these dotfiles you can find them at https://github.com/AmilieCoding/dotfiles or if you want to make your own theme based off the colour scheme only you can check the colour theme out at https://github.com/catppuccin/catppuccin, the specific theme is called Mocha!
