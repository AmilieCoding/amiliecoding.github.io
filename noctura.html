Noctura Client Installer


Noctura client, developed by Flaily can be found at the link below. It will download a .jar installer which you may run depending on your operating system.

If you are on Windows and have java installed, simply double click the .jar. On macOS and Linux you should install Java, navigate to the jar's location in terminal with cd and type `java --jar [jarname]`

You can find the source code for Noctura at https://github.com/clueeng/Flauxy2
